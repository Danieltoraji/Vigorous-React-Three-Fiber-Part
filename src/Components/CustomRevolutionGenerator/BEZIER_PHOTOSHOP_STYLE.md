# 🎨 Photoshop 风格贝塞尔曲线工具

## ✨ 专业版贝塞尔曲线编辑器

现在的贝塞尔模式已经升级为专业的矢量绘图工具，支持像 Photoshop 钢笔工具一样精确控制曲线！

## 🎯 核心功能

### 1. **添加锚点**
- 在画布上点击即可添加锚点（红色圆点）
- 至少需要 2 个锚点才能形成曲线
- 每个锚点自动带两个控制手柄

### 2. **拖动手柄调整曲率**
- **红色虚线**：控制手柄
- **红色小圆点**：手柄端点
- 拖动手柄端点可以调整曲线的曲率和方向
- 左手柄控制"进入"方向，右手柄控制"离开"方向

### 3. **移动锚点位置**
- 直接拖拽锚点（红色圆点）可以改变其位置
- 手柄会跟随锚点一起移动
- 实时看到曲线形状变化

## 📖 使用步骤

### 第一步：绘制基础形状
1. 切换到"🎨 贝塞尔"模式
2. 在画布上点击添加 2-4 个锚点
3. 系统自动生成平滑的三次贝塞尔曲线

### 第二步：精细调整
1. **调整曲率**：拖动锚点的红色手柄
   - 向左/右拖动：改变曲线弯曲程度
   - 向上/下拖动：改变曲线倾斜角度
   
2. **调整位置**：拖动锚点本身
   - 改变锚点在画布上的位置
   - 曲线会自动适应新位置

3. **实时预览**：所有调整立即反映在曲线上

### 第三步：完成绘制
- 系统自动将贝塞尔曲线转换为密集点阵
- 用于后续的 3D 模型生成

## 🔧 技术细节

### 数据结构
```javascript
// 每个锚点的完整结构
{
  x: 100,        // 锚点 X 坐标
  y: 75,         // 锚点 Y 坐标
  cp1: {         // 左手柄（控制曲线进入方向）
    x: 80,
    y: 75
  },
  cp2: {         // 右手柄（控制曲线离开方向）
    x: 120,
    y: 75
  }
}
```

### 三次贝塞尔曲线公式
```
B(t) = (1-t)³·P0 + 3(1-t)²·t·CP0 + 3(1-t)·t²·CP1 + t³·P1

其中:
- P0: 起点锚点
- P1: 终点锚点  
- CP0: 起点的右手柄 (cp2)
- CP1: 终点的左手柄 (cp1)
- t: 0 到 1 的参数
```

### 智能手柄初始化
- 新锚点的手柄默认沿切线方向
- 如果有前一个锚点，自动计算最优手柄方向
- 默认手柄长度：30 像素

## 💡 高级技巧

### 1. **创建尖锐转折**
- 将两个相邻锚点的手柄设置为不同角度
- 可以创建 S 形曲线或尖锐转折

### 2. **对称曲线**
- 手动将左右手柄调整为对称位置
- 获得完美的对称曲线段

### 3. **平滑过渡**
- 让相邻锚点的手柄保持在同一直线上
- 获得 G1 连续的平滑过渡

### 4. **闭合形状**
- 虽然当前版本不支持闭合
- 但可以通过首尾相连近似实现

## 🚀 实际案例

### 案例 1：花瓶轮廓
1. 添加 3 个锚点形成基本轮廓
2. 调整顶部锚点的左手柄向上，形成瓶口
3. 调整中间锚点的右手柄向右，形成瓶身膨胀
4. 调整底部锚点的左手柄向下，形成瓶底收缩

### 案例 2：波浪曲线
1. 添加 5 个等距锚点
2. 交替调整手柄向上/向下
3. 形成优美的正弦波形

### 案例 3：心形上半部分
1. 添加 3 个锚点（左、中、右）
2. 左边锚点：右手柄向右上
3. 中间锚点：左手柄向左上，右手柄向右上
4. 右边锚点：左手柄向左上
5. 调整手柄长度获得完美心形

## ⚠️ 注意事项

1. **最少锚点数**：至少 2 个锚点才能生成曲线
2. **手柄可见性**：只有选中锚点时才显示手柄（未来优化）
3. **性能考虑**：建议不超过 10 个锚点，避免过于复杂
4. **数据输出**：曲线会被转换为 50 点/段的密集点阵
