# 异形生成器使用指南

## 🎨 三种绘制模式

现在您可以在画布上使用**三种**不同的绘制模式来创建自定义 3D 模型：

### 1. **点绘模式**（✏️ 传统模式）
- 点击画布添加独立的点
- 系统自动连接相邻点形成折线
- 适合创建有棱角的几何形状、齿轮、多边形等

### 2. **贝塞尔曲线模式**（🎨 精确控制）
- 点击画布添加控制点
- 系统自动生成平滑的二次贝塞尔曲线
- 支持拖拽调整最后一个控制点的位置
- 适合需要精确控制的平滑曲线、工业造型

### 3. **自由绘制模式**（🖌️ 手绘体验） ⭐ NEW!
- **按住鼠标左键并拖动**，像用铅笔一样直接画曲线
- 系统自动对绘制的路径进行平滑处理
- 实时预览曲线效果
- 最自然、最直观的绘制方式
- 适合有机形态、艺术造型、随意涂鸦

## 🔄 模式切换

点击模式按钮在三种模式间循环切换：
```
✏️ 点绘 → 🎨 贝塞尔 → 🖌️ 自由绘制 → ✏️ 点绘 → ...
```

当前激活的模式会以不同颜色高亮显示：
- 点绘模式：蓝色背景
- 贝塞尔模式：蓝色背景
- 自由绘制：**紫色渐变背景**（特别标识）

## 📝 使用步骤

### 轮廓曲线绘制（必需）

#### 方法一：自由绘制（推荐新手）
1. 切换到"🖌️ 自由绘制"模式
2. **按住鼠标左键不松手**
3. 在画布上拖动绘制你想要的轮廓
4. 松开鼠标完成绘制
5. 系统自动平滑处理曲线

#### 方法二：贝塞尔曲线（推荐专业用户）
1. 切换到"🎨 贝塞尔"模式
2. 依次点击添加 2 个或更多控制点
3. 观察实时生成的平滑曲线
4. 至少需要 2 个点才能生成有效曲线

#### 方法三：点绘模式（推荐精确点位）
1. 切换到"✏️ 点绘"模式
2. 在关键位置逐个点击添加点
3. 系统用直线连接所有点

### 路径曲线绘制（可选）
1. 如果需要创建扫掠体（非旋转对称），在右侧画布绘制路径曲线
2. 同样支持三种绘制模式
3. 路径曲线决定轮廓沿什么轨迹延伸
4. 不绘制路径曲线时，默认生成旋转体

## 💡 技巧与提示

### 自由绘制模式最佳实践
- ✅ **匀速移动**：保持鼠标移动速度均匀，获得更好的平滑效果
- ✅ **一笔成型**：尽量一次性绘制完整曲线，避免断断续续
- ✅ **适度细节**：不需要过于精细的细节，系统会自动平滑
- ✅ **起点终点**：绿色点表示起点，黄色点表示终点
- ⚠️ **点数优化**：绘制完成后系统会保留关键点，去除冗余点

### 贝塞尔曲线最佳实践
- **控制点数量**：建议 3-5 个控制点即可获得平滑曲线
- **点间距**：控制点之间保持适当距离，避免过密
- **起始点**：第一个和最后一个控制点会显示为红色，中间点为黄色
- **实时调整**：添加新点时可以拖动调整位置

### 点绘模式最佳实践
- **关键特征点**：在曲线的转折点、端点等关键位置点击
- **简洁优先**：用最少的点表达形状，避免过多细节

### 模式选择建议
- 🖌️ **自由绘制**：快速原型、有机形态、艺术创作
- 🎨 **贝塞尔曲线**：精确设计、工业造型、规则曲线
- ✏️ **点绘**：几何图形、多边形、硬表面建模

### 方程生成（保留功能）
- 仍然可以使用数学方程快速生成曲线
- 在输入框中输入 JavaScript 数学表达式
- 按回车或点击"生成"按钮执行

## 🔧 技术细节

### 自由绘制算法
- 使用 **Catmull-Rom 样条插值**实现平滑
- 张力系数设置为 0.5，平衡平滑度和保真度
- 在相邻点之间自动插入中间点
- 使用二次贝塞尔曲线连接所有点

### 贝塞尔曲线算法
- 使用二次贝塞尔曲线插值
- 自动在控制点之间生成 50+ 个采样点
- 确保导出的 3D 模型足够平滑

### 数据格式
```javascript
// 自由绘制/点绘模式 - 密集点阵
[
  { x: 100, y: 75 },
  { x: 102, y: 74 },
  { x: 105, y: 73 },
  // ... 更多点
]

// 贝塞尔模式 - 控制点
[
  { x: 100, y: 75 },   // 控制点 1
  { x: 150, y: 50 },   // 控制点 2
  { x: 200, y: 100 }   // 控制点 3
]
```

### 3D 生成逻辑
- **旋转体**：仅轮廓曲线 → 绕 Y 轴旋转生成
- **扫掠体**：轮廓曲线 + 路径曲线 → 沿路径扫掠生成

## ⚠️ 注意事项

1. **最少点数**：
   - 自由绘制：至少 2 个点
   - 贝塞尔模式：至少 2 个控制点
   - 点绘模式：至少 2 个点

2. **坐标系统**：画布原点 (0,0) 在左上角

3. **对称轴**：红色垂直线为旋转轴（X=中心）

4. **清空操作**：点击"清空"按钮会清除所有点和控制点

5. **模式切换**：切换模式不会清除已绘制内容，但清空会重置所有模式的数据

## 🎯 示例场景

### 使用自由绘制模式 🖌️
- ✅ 快速勾勒花瓶轮廓
- ✅ 绘制随意的艺术装饰品
- ✅ 创建有机形态的雕塑
- ✅ 徒手画草图验证想法

### 使用贝塞尔曲线模式 🎨
- ✅ 设计精确的工业零件
- ✅ 创建流线型汽车外壳
- ✅ 制作标准圆弧和曲线
- ✅ 需要 CAD 级别精度的场景

### 使用点绘模式 ✏️
- ✅ 创建齿轮、螺栓等多边形
- ✅ 设计金字塔、棱柱等几何体
- ✅ 制作有明确转折点的硬表面模型

## 🚀 快捷操作

### 自由绘制
- **按住拖动**：连续绘制曲线
- **松开鼠标**：完成绘制

### 贝塞尔曲线
- **单击**：添加新控制点
- **拖动**：调整最新添加的点位置

### 点绘
- **单击**：添加新点

### 通用
- **清空**：一键清除所有点
- **方程生成**：快速创建规则曲线

---

**提示**：无论使用哪种模式，系统最终都会将曲线转换为密集的点阵数据用于 3D 几何体生成，确保模型精度。自由选择最适合您需求的绘制方式！
